# 🚀 Aegis AI

**AI-Powered Project Management Dashboard**

---

## ✨ Features

- ✅ **Authentication** – Signup, Login, Forgot Password
- ✅ **Company Management** – Create/Join company with invite code
- ✅ **Projects** – Add, Edit, Delete, Health/Risk tracking
- ✅ **Risks** – Add, Delete, Severity levels
- ✅ **Team Management** – Add/Remove team members
- ✅ **AI Insights** – CEO/Manager/Dev perspectives with Gemini AI
- ✅ **Audit Log** – Track all activities
- ✅ **Role-Based Access** – CEO, Manager, Developer roles
- ✅ **RLS Security** – Company data isolation
- ✅ **Responsive Design** – Works on all devices

---

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| React 18 | Frontend UI |
| Supabase | Database + Authentication |
| Gemini AI | AI-powered insights |
| Chart.js | Analytics charts |
| jsPDF | PDF reports |
| GitHub Pages | Hosting |
